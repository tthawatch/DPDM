# Types of Data sets
## Record Data
1. relational records เป็นการใช้ Index หรือ ID ใช้เชื่อมข้อมูลจากตารางหนึ่งไปยังอีกตารางหนึ่ง เช่น เลขบัตรประชาชน รหัสนักศึกษา
2. data matrix,e.g,crosstabs เป็นการเก็บข้อมูลที่หัวข้อ 2 ฝั่ง
3. transaction data เป็นการเก็บข้อมูลที่ไม่มีหัวข้อ 
4. document data : Term-frequency vector (matrix) of text documents เป็นการเก็บข้อมูลที่มี attributes ด้านบน และตัวบอกตัวตน้านข้าง

## Improtant Characteristics of structered Data
1. Dimensionality เป็นการเชื่อมต่อกันแต่ละมิติ
2. Sparsity จุดที่ม่มีข้อมูลจะเป็นช่องว่าง
3. Resolution เป็นเรื่องเกี่ยวกับรายละเอียดภาพที่ต้องปรับสเกลให้เท่ากัน
4. Distribution อธิบายข้อมูลค่ากลางและค่าารกระจาย
## Data Objects
Data Objects อธิบายมาจาก attributes
database rows >> data objects ; columns >> attributes
# Attributes
1. Nominal ให้ชื่อแทนข้อมูล ไม่มีความหมายทางคณิตศาสตร์
2. Binary ข้อมูลที่มี 2 ค่าเท่านั้น (0,1) 
2.1 ข้อมูลสมมาตร 
2.2 ข้อมูลไม่สมมาตร
3. Ordinal ข้อมูลเรียงลำดับได้ (เล็ก กลาง ใหญ่) 
4. Numeric ข้อมูลที่เป็นตัวเลข
4.1 Interval ไม่มีศูนย์แท้ เป็นค่าที่นิยามขึ้น
4.2 ratio มีศูนย์แท้
4.3 Discrete ไม่ต่อเนื่อง
4.4 Continous ต่อเนื่อง
